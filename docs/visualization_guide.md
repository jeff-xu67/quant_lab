# 可视化图表解读指南

```bash
# 运行回测并自动打开浏览器报告
python scripts/run_backtest.py configs/default.yaml --show-report
```

```python
# 或在代码中手动生成
import polars as pl
from quant_lab.metrics.visualization import (
    plot_equity_curve,
    plot_drawdown,
    plot_monthly_returns,
    plot_trade_analysis,
    create_report,

trades = pl.read_parquet("results/<run_id>/trades.parquet")

```

## 1. 权益曲线 (Equity Curve)

**作用：** 展示策略资金随时间的增长情况，是评估策略最直观的图表。

**怎么看：**

- **Y 轴**：归一化后的累计收益（1.0 = 初始资金，1.2 = 赚了 20%）
- **X 轴**：时间
- **实线**：策略表现
- **虚线**（如有）：基准（如沪深300）

**判断好坏：**

| 形态 | 含义 |
|------|------|
| 稳定向上的斜线 | 策略稳定盈利，波动小 |
| 锯齿形上升 | 盈利但波动大，持有体验差 |
| 长期横盘后突然拉升 | 策略依赖少数大行情，平时不赚钱 |
| 持续低于基准虚线 | 跑不赢买入持有，策略无效 |

**关注点：**
- 曲线是否始终在 1.0 上方（是否亏过本金）
- 与基准对比：如果大部分时间跑不赢沪深300，策略就没有存在的意义
- 斜率是否稳定：越平滑说明收益越可预期

---

## 2. 回撤图 (Drawdown)

**作用：** 展示策略从历史最高点下跌了多少，衡量最坏情况下你要承受的亏损。

**怎么看：**

- **Y 轴**：回撤百分比（负值），0% 表示处于历史新高
- **X 轴**：时间
- **红色填充区域**：当前回撤幅度

**判断好坏：**

| 指标 | 优秀 | 一般 | 危险 |
|------|------|------|------|
| 最大回撤 | < 10% | 10-20% | > 30% |
| 回撤持续时间 | < 3个月 | 3-12个月 | > 1年 |

**关注点：**
- **深度**：谷底越深，说明极端情况下亏损越大。-30% 意味着 100 万最多会缩水到 70 万
- **宽度**：回撤持续越久，说明策略恢复能力越差。超过 1 年的回撤在实盘中很难坚持
- **频率**：频繁触底说明策略稳定性差

**实际意义：** 最大回撤是你实盘运行这个策略时，心理上需要准备承受的最大亏损。如果你无法接受 -30% 的回撤，就不应该使用该策略。

## 3. 月度收益热力图 (Monthly Returns)

**作用：** 以年份×月份的矩阵形式展示每个月的收益率，快速发现季节性规律和异常月份。

**怎么看：**

- **行**：年份
- **列**：月份 (Jan-Dec)
- **颜色**：绿色 = 正收益，红色 = 负收益，颜色越深幅度越大
- **数字**：该月收益百分比

**判断好坏：**

| 形态 | 含义 |
|------|------|
| 大面积浅绿 | 稳定小赚，低波动 |
| 绿红交替但绿多 | 正常盈利策略 |
| 某些月份持续深红 | 可能有季节性风险（如每年1月亏损） |
| 某一年全行深红 | 策略在某类行情下完全失效 |

**关注点：**
- 是否有连续多个红色月份（连续亏损期）
- 是否有某个月特别深红（单月大亏，可能是黑天鹅）
- 盈利是否均匀分布，还是集中在少数月份

---

## 4. 交易分析 (Trade Analysis)

**作用：** 展示每笔交易的金额分布和手续费消耗情况。

**怎么看：**

- **左图：交易金额分布** — 每笔卖出的成交金额直方图
- **右图：手续费分布** — 每笔交易的手续费直方图

**关注点：**
- 单笔交易金额是否合理（过小说明仓位太分散，过大说明集中度高）
- 手续费总量占收益的比例（如果手续费吃掉大部分利润，策略就不可行）

---

## 5. 滚动 Sharpe (Rolling Sharpe, 60天)

**作用：** 展示策略风险调整收益随时间的变化，判断策略是否在衰退。

**怎么看：**

- **Y 轴**：60天窗口的年化 Sharpe 比率
- **X 轴**：时间
- **> 0**：策略在该时段内风险调整后是赚钱的
- **< 0**：策略在该时段内亏损

**判断好坏：**

| Sharpe 值 | 含义 |
|-----------|------|
| > 2.0 | 非常优秀（少见） |
| 1.0 - 2.0 | 优秀 |
| 0.5 - 1.0 | 可接受 |
| 0 - 0.5 | 勉强 |
| < 0 | 亏损 |

**关注点：**
- 曲线是否持续走低（策略在失效/Alpha衰减）
- 是否频繁跌入负值区域
- 近期表现是否比早期差（过拟合的信号）

## 6. 综合报告面板 (Full Report)

`create_report()` 生成一个包含 6 个子面板的综合图表：

```
┌─────────────────────┬─────────────────────┐
│   权益曲线           │   指标汇总表         │
├─────────────────────┼─────────────────────┤
│   回撤图            │   月度收益热力图      │
├─────────────────────┼─────────────────────┤
│   交易分布           │   滚动 Sharpe        │
└─────────────────────┴─────────────────────┘
```

一张图看全貌，适合策略初步筛选和对比。

## 核心指标速查

| 指标 | 公式/含义 | 越高越好? |
|------|-----------|-----------|
| Total Return | (期末资金 / 初始资金) - 1 | 是 |
| Annualized Return | 几何年化收益率 | 是 |
| Max Drawdown | 最大峰谷跌幅 | 否（越接近0越好） |
| Sharpe Ratio | (年化收益 - 无风险利率) / 年化波动率 | 是 |
| Sortino Ratio | 只考虑下行波动的 Sharpe | 是 |
| Calmar Ratio | 年化收益 / 最大回撤 | 是 |
| Win Rate | 盈利交易数 / 总交易数 | 是 |
| Profit Factor | 总盈利 / 总亏损 | 是（>1才赚钱） |
| Volatility | 收益率的年化标准差 | 否（越低越稳） |

## 实用技巧

**交互操作（Plotly图表）：**
- 鼠标悬停：查看具体数值
- 框选放大：拖动选择区域放大查看
- 双击：恢复原始视图
- 图例点击：显示/隐藏某条线

**对比多个策略：**

```python
from quant_lab.experiment.comparison import RunComparison

comp = RunComparison("results")
runs = comp.list_runs()
print(runs)  # 查看所有 run_id

# 叠加权益曲线对比
fig = comp.plot_equity_comparison(["run_id_1", "run_id_2", "run_id_3"])
fig.show()
```

**保存图表：**

```python
fig.write_html("my_report.html")       # 交互式 HTML
fig.write_image("my_report.png")       # 静态图片 (需安装 kaleido)
```
